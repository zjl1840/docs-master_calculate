## 1 移动端概述

!!! Abstract ""
    仪表板除了可以在 PC 端查看外，还可以在移动端查阅，满足用户随时随地查阅报表的需求。  
    **请注意：访问移动端无需安装 APP，使用浏览器直接访问 DataEase 链接信息即可。**
