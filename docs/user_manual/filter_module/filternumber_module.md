## 1 数字下拉组件

!!! Abstract ""
	可通过与视图组件的数值型字段绑定，点击数字下拉组件，即弹出下拉选项框，选定下拉框内的数值，过滤视图组件的数据。

## 2 数字列表组件

!!! Abstract ""
	可通过与视图组件的数值型字段绑定，数字列表组件以列表的形式展示字段内数值，用户可通过勾选列表内的数值，过滤视图组件的数据。
